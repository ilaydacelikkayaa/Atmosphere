##  Overview / Genel Bakış
**English:** Atmosphere is a sophisticated iOS application that bridges the gap between environmental conditions and emotional resonance. By leveraging **Google Gemini AI**, the app analyzes current weather data alongside user-provided moods to curate a personalized musical journey via the **iTunes Search API**.

**Türkçe:** Atmosphere, çevresel koşullar ile duygusal dışavurum arasındaki bağı kuran gelişmiş bir iOS uygulamasıdır. **Google Gemini AI** desteğiyle, anlık hava durumu verilerini ve kullanıcı modunu analiz ederek **iTunes Search API** üzerinden kişiselleştirilmiş bir müzik yolculuğu sunar.

---

##  Key Features / Öne Çıkan Özellikler
* **AI Atmosphere Analysis:** Uses NLP to interpret user feelings and weather patterns for precise music matching.
* **Programmatic UI:** Built entirely without Storyboards for maximum flexibility and performance.
* **Atmosphere Diary:** A "Polaroid-style" collection view to persist your favorite musical moments using **UserDefaults**.
* **Modern Interaction:** Implements iOS **Context Menus** for intuitive data management and deletion.
* **High-Res Visuals:** Adaptive typography and high-resolution artwork loading for a premium user experience.



---

##  Tech Stack / Teknik Yetkinlikler
* **Language:** Swift 5.10+
* **Framework:** UIKit (Programmatic Layout)
* **Architecture:** MVVM (Model-View-ViewModel)
* **APIs:** Google Gemini AI, OpenWeatherMap, iTunes Search API
* **Data Persistence:** JSON Encoding/Decoding with UserDefaults
